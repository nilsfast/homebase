homebase is a server application for managing and documenting your homelab assets. It provides a web interface to keep track of your hardware, software, and network configurations, making it easier to maintain and organize your homelab environment.

## Features
- Asset management: Keep track of your hardware components, software installations, and network configurations.
- (TODO) Documentation: Create and maintain documentation for your homelab setup, including processes and configurations. This can be useful helping others understand your setup, or for your future self when you forget the details.
- Web interface: Access your homelab information from any device with a web browser.
- API: Integrate homebase with other tools and services in your homelab ecosystem.
- (TODO) CLI tool: Manage your homelab assets and documentation from the command line.

## Installation
You can install homebase using pip:
```bash
pip install homebase-server
```

## Usage
After installing homebase, you can start the server with the following command:
```bash
homebase serve
```
This will start the homebase server, and you can access the web interface by navigating to `http://localhost:8000` in your web browser.
